# Support

Use the channel that matches the kind of help you need:

- Bug reports: open a GitHub issue with reproduction steps.
- Feature requests: open a GitHub issue describing the use case and expected behavior.
- Security reports: use the private advisory flow described in [SECURITY.md](./SECURITY.md).

For general usage questions, include the output of:

```sh
codeye version
codeye doctor
```

That usually answers the first round of environment questions immediately.
