What it does

Everything you need,
nothing you don't.

Built for engineers who want answers in milliseconds, not megabytes of tool overhead.

⚡

Sub-100ms scans

Parallel goroutine pool + msgpack cache. Fast on first run, instant on repeat. No index to maintain.

🔍

Line-level breakdown

Code, comments, and blank lines counted separately per language. Not just totals — actual signal.

👤

Git blame rollup

Per-author line ownership across the whole repo, not just one file. See who owns what, fast.

📈

Growth history

Week-by-week LoC growth with sparklines. Watch your codebase evolve over time.

🎯

Hotspot detection

Files with the most churn relative to size. Find the code that changes the most.

📤

Multiple formats

Table, JSON, CSV, Markdown, badge — pipe into scripts, CI, or dashboards. Your data, your pipeline.

Quick start

Up in 30 seconds.

1

Install

$ go install github.com/codeye/codeye/cmd/codeye@latest

Or download a pre-built binary from Releases.

2

Scan a repo

$ cd ~/projects/myapp
$ codeye

Run inside any git repository. No flags required.

3

Go deeper

# Blame by author
$ codeye --blame

# Week-by-week growth
$ codeye --history

# Most-changed files
$ codeye --hotspot

# Export JSON for CI
$ codeye --format json
