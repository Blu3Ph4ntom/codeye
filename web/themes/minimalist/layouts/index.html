Get Started View on GitHub

⚡ Performance

Scans 10k files in under 100ms. Local caching with msgpack ensures your next scan is nearly instantaneous.

📦 Native Porcelain

No custom git implementations. We use git ls-files --porcelain and git blame --porcelain directly from the source.

📊 Rich Output

From terminal tables and sparklines to JSON, CSV, and Markdown badges. Built for both humans and CI/CD pipelines.

🧩 Monorepo Ready

Full support for Go 1.22+ workspaces. Handles massive repositories without breaking a sweat.
